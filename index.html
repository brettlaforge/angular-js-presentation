Installation

  1. Download AngularJS (http://angularjs.org/)
  2. Include the AngularJS Script
  3. Add the 'ng-app' attribute
HTML 
<!doctype html>
<html lang="en" ng-app>
	<head>
		<script src="js/angular.min.js"></script>
	</head>
	<body>
	</body>
</html>

Simple Example

HTML 
<!doctype html>
<html lang="en" ng-app>
	<head>
		<script src="js/angular.min.js"></script>
	</head>
	<body>
		<input type="text" ng-model="name">
		<p ng-bind="name"></p>
		{{ name }}
	</body>
</html>
Output

{{ name }}

AngularJS is a

HTML Compiler

Directives

  • Attribute, Class, Comment, or Element
  • Sprinkled in your DOM
  • Extend built in behaviour
  • Add custom behaviour
  • Transform the DOM
  • Angular comes with a built set of directives
    Usually prefixed with 'ng'
  • Define your own directive to add custom behaviour

Directives

Directives can be invoked in many ways:

Attribute

<span my-directive></span>

Class

<span class="my-directive"></span>

Comment

<!-- directive: my-directive -->

Element

<my-directive></my-directive>

Directives

Directives can be invoked in many ways:

Attribute

<span ng-bind="name"></span>
<span ng_bind="name"></span>

Attribute Namespace

<span ng:bind="name"></span>

Attribute Prefix

<span data-ng-bind="name"></span>
<span x-ng-bind="name"></span>

'ng-app' Directive

Starts Angular

Defines Angular Boundary

Directives and Expressions will only be processed inside a 'ng-app' Directive

Gives you the freedom to choose:

  • Entire Page
  • Only Portion of Page
  • Multiple 'ng-app' Directives on the Same Page

DOM Templates VS String Templates

DOM Templates
Bindings and Control Flow (each, if, else, etc) go into the DOM
String Templates
Build up a string and inject it into the DOM (innerHtml). When values change rebuild string and inject it again.

Services

  • Are singletons that carry out specific common tasks
  • Built in services are prefixed with '$'
  • $http - Low level access to the browser's XMLHttpRequest object (AJAX)
  • $window - Low level access to the browser's window object

Dependecy Injection

  • Allows a piece of code to get hold of its dependencies
